int Search::Plan ( )

Plan an initial path from start to goal, or if any cost changes are detected since it was last called (any calls to ChangeCost()) then it replans. The generated path can be obtained by following the next pointers from start vertex until goal vertex is reached

Returns
total number of vertices along the path
